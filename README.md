霓虹蛇 Neon Serpent

一款赛博朋克风格的贪吃蛇进阶游戏，共8关，难度递增。

试玩链接：
https://wea3qtg9.mule.page/

玩法核心：

在不规则的霓虹迷宫中控制蛇收集指定颜色的食物达到通关条件，同时躲避红砖墙、炸弹和AI蛇的威胁，每关有时间限制。

关卡设计：

第一关仅需收集2种颜色食物，1条AI蛇，90秒时限
最终关需收集8种颜色各8个，5条AI蛇，20颗炸弹，200秒时限
每关地形随机生成：不规则边界 + 岩石集群，越往后障碍越密
特殊机制：

护盾（地图常驻x2）— 抵挡2次碰撞
冻结AI（地图常驻x2）— 冻结全部AI蛇6秒
双倍分（地图常驻x2）— 8秒内得分翻倍
缩身食物（地图常驻x3）— 蛇身回归初始长度，便于穿行
随机掉落：复活心、+5分、减速
操作：

方向键/WASD移动，空格暂停，ESC重开。移动端支持触屏方向键和滑动。

技术特点：

纯前端实现，单HTML文件，Canvas渲染，Web Audio音效，无外部依赖，浏览器直接运行。
